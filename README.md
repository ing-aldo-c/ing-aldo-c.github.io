# 🗺️ CV Geospaziale Interattivo — Ing. Cataldo Carletta

Curriculum Vitae in formato **web interattivo e geolocalizzato**, realizzato con HTML, CSS e Leaflet.js.

## 🔗 Demo live
👉  [ing-aldo-c.github.io](https://ing-aldo-c.github.io/)

---

## 📋 Descrizione

Un CV non convenzionale che presenta le esperienze professionali e il percorso formativo
su una **mappa interattiva geospaziale**, con marker posizionati nelle città reali
dove ogni attività si è svolta.

---

## ✨ Funzionalità

- 🗺️ **Mappa interattiva** con Leaflet.js — vista Satellite e Street
- 📍 **Marker geolocalizzati** per ogni esperienza lavorativa e titolo di studio
- 🔵 **Filtro per categoria** — mostra/nasconde Esperienze o Formazione
- 📱 **Design responsive** — ottimizzato per mobile e desktop
- 🌙 **UI dark mode** con stile glassmorphism
- 🔗 **Link diretto** a LinkedIn e contatto email

---

## 🛠️ Tecnologie utilizzate

| Tecnologia | Utilizzo |
|---|---|
| HTML5 / CSS3 | Struttura e layout |
| [Leaflet.js](https://leafletjs.com/) | Mappa interattiva |
| CartoDB / ESRI | Tile layer Satellite e Street |
| Font Awesome 6 | Icone |
| Google Fonts (Inter) | Tipografia |
